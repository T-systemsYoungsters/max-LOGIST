```python
    sum=0
    for i in list:
        sum = sum + i
    return sum
list = [45, 2, 10, -5, 100]
print(sum_list(list))
```
6.<br>
um das P auch umzudrehen muss man das "i*-1" noch -1 nehmen<br>
```python
def reverse(text):
    result = ""
    text_length = len(text)
    for i in range(text_length):
        result = result + text[(i * -1)-1]
    return result
text = "Programming is the coolest thing ever."
print(reverse(text))
```
